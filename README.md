# GitMind 🔍
> AI-powered Git commit analyzer — understand any GitHub repo instantly.

![Status](https://img.shields.io/badge/status-in%20development-yellow)
![Stack](https://img.shields.io/badge/stack-Next.js%20%7C%20FastAPI%20%7C%20Gemini%20%7C%20GitHub%20API-blue)

## What is GitMind?
Paste any public GitHub repo URL → GitMind fetches commit history via GitHub API → AI explains each commit in plain English → detects bug-introducing commits → generates visual changelog.

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | Next.js 14, Tailwind CSS, shadcn/ui, Framer Motion |
| Backend | Python, FastAPI, Gemini API |
| Data | GitHub REST API v3 |
| Deployment | Vercel (frontend), Render (backend) |
